Part 02 — The Resurrection
Backpropagation

How a network learns from its mistakes

The Architecture
A network of neurons

Layers of nodes connected by weighted edges. Each weight controls how strongly one neuron influences the next.
The network starts knowing nothing.

The Architecture
Layers of weighted edges

The network has an input layer, hidden layers, and an output layer. Information flows layer by layer, shaped at each step by the weights.

Forward Pass
Step 1 — Forward pass

Data enters at the left and flows right. Each neuron multiplies its inputs by weights, sums them, and fires a signal forward.

Prediction
The network makes a guess

Output neurons produce a probability distribution. The network is confident — but it may be confidently wrong. That is where learning begins.

Loss Function
Step 2 — Measure the error

The loss function computes how wrong the prediction was. A big gap means a big penalty. The goal: drive this number to zero.

Mathematics
The chain rule

Calculus tells us: if we know how the loss changes at the output, we can trace responsibility backwards through every layer.

∂L/∂w = ∂L/∂a · ∂a/∂w
Backpropagation
Step 3 — Backward pass

The error signal flows right to left. Each connection receives a "blame score" — its gradient — measuring how much it contributed to the mistake.

Gradient Descent
Step 4 — Update weights

Every weight is nudged by gradient descent: move a tiny step in the direction that reduces the error.

w ← w − η · ∇ w
Training Loop
Repeat thousands of times

Each cycle through the data is called an epoch. Forward pass, measure error, backward pass, update weights.

Complete
That's backpropagation

Forward → Loss → Backward → Update → Repeat.

The engine behind every modern AI system.
